unsigned int iris::dist500::updateSensorConnectionsViaBroadcast ( )

fillLookup Uses sensorList to fill the lookupSerialToSensorInstance container.

Queries config parameter "serial" from each sensor

Parameters
sensorListUpdates instances in lookupSerialToSensorInstance by using a broadcast. Can be used with all protocols, but is only useful for protocols where addresses may change, such as with DHCP. Call when you are sure, the devices have all rebooted and are responsive.
Returns
Returns number of update sensors that respnded. Serves as indicator for sensors finishing booting and address assignments.
