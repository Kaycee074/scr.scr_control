virtual

Signals the availability of new data within the attached source.

This callback method is abstract. Derived PictureSink classed therefore need to implement it.

Parameters
picturePicture instance containing the new data
pictureIndex0-based index of the picture if the total number of pictures is known, 0 otherwise
pictureIndexCountTotal number of picture (0 if unknown)
