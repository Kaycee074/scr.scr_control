void iris::drivers::Sensor::setUipAddress ( unsigned int  uipAddress)

setUipAddress Changes the UIP address.

The UIP address is the number after the hash sign in a URL.

Parameters
uipAddress
Note
If the UIP changes, it must be updated, otherwise messages will not be received and will be discarded.
The UIP address can change on grouping or ungrouping of sensors. Direct configuration of the UIP is another possibility.
