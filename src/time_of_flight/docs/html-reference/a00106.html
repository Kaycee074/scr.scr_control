Detailed Description

The HttpRequest class.

See RFC 2616 for info on Hypertext Transfer Protocol – HTTP/1.1 Terms used here from that RFC: Request-Line, general-header, request-header, entity-header, message-body, CRLF, SP, Method, Request-URI, HTTP-Version
