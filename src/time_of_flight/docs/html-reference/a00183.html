Detailed Description

Represents a group of DIST500 sensors (eg a function area).

Sensors are usually grouped for wide doors.

Member Function Documentation

void iris::dist500::SensorGroup::addSensor ( Dist500 *  sensor)

adds a sensor.

The first sensor is added as the "master" sensor, others are added as slaves. The master sensor is responsible for sending total count results.
