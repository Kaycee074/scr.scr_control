22 // Socket CAN implementation of our CAN driver.
23 //
24 // BAUD RATE NOTES
25 //
26 // There is no way to set the baud rate via SocketCAN, so a supported baud rate
27 // must be configured for the interface.
28 // Currently the Irma Matrix supports 125000 baud.
29 // Example shell commands (sudo may be necessary):
30 // ip link set can0 down
31 // ip link set can0 type can bitrate 125000
32 // ip link set can0 up
33 //
34 // Permanent configuration of bitrate is possible in /etc/network/interfaces
