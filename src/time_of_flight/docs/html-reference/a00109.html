Detailed Description

The HttpServer class.

This class allows to serve contents from the local machine. The HttpServerSettings are passed as a reference in the constructor. It can be useful to also pass a callback for status messages, so the HttpServer's status can be observed.

Note
Internally uses HttpRequest, HttpResponse, HttpConnectionHandler. There is no need for using them as standalone.
ResponseDataCache is also useful for caching via other non-ethernet protocols like CAN.
